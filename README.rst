Install
------------------------------------------------------------------------------

``learn_tantivy`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-tantivy

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-tantivy
